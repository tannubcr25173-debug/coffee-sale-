# Coffee Shop Sales Visualization

## Project Overview

This project analyzes and visualizes coffee shop sales data using
**NumPy**, **Pandas**, **Matplotlib**, and **Seaborn**. It organizes
sales information, calculates revenue, and presents business insights
through multiple charts and graphs.

------------------------------------------------------------------------

## Objective

Practice the following concepts:

-   NumPy Arrays
-   Pandas DataFrame
-   Data Visualization
-   Business Data Analysis
-   Statistical Calculations

------------------------------------------------------------------------

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

### Libraries Used

-   NumPy
-   Pandas
-   Matplotlib
-   Seaborn

------------------------------------------------------------------------

## Features

-   Create NumPy arrays for coffee sales data.
-   Build a Pandas DataFrame.
-   Calculate revenue for each coffee item.
-   Generate:
    -   Bar Chart
    -   Pie Chart
    -   Line Chart
    -   Histogram
    -   Scatter Plot
    -   Seaborn Bar Plot
    -   Box Plot
    -   Correlation Heatmap
-   Display business insights such as:
    -   Most Sold Coffee
    -   Most Expensive Coffee
    -   Average Coffee Price
    -   Total Revenue

------------------------------------------------------------------------

## How to Run

1.  Open the notebook in Jupyter Notebook or Google Colab.

2.  Install required libraries if needed:

    ``` bash
    pip install numpy pandas matplotlib seaborn
    ```

3.  Run all notebook cells.

4.  View the generated visualizations and analysis.
